## Getting started

#### Requirements

- `yarn` (or `npm`)
- `gulp` (`yarn global add gulp-cli` or `npm i -g gulp-cli`)

#### Installation

```sh
git clone https://github.com/jclerc/greenspot.git
cd greenspot
yarn # or `npm i`
gulp modernizr
```

Then to make production build, use `gulp build`.
To start development mode with live reload, simply run `gulp`.
